Rest API for managing drills, categories, and instructions.

Drill Controller

Offers all CRUD operations for Drills in the database.

Find a Drill by its ID.

Search to see if a Drill exists by the given ID.

path Parameters
id
required
integer <int64>

Responses

Generated server url

http://localhost:5433/drill/id/{id}

Update a Drill by its ID.

Update a drill's contents (category, instructions, etc.) by its ID. Returns the newly updated Drill.

path Parameters
id
required
integer <int64>
Request Body schema: application/json
required
name
required
string [ 1 .. 255 ] characters

Existing or new name of the Drill

Array of objects (Instructions)

List of different instructions to perform this drill.

categories
Array of integers <int64> [ items <int64 > ]

List of Category IDs the Drill belongs to.

sub_categories
Array of integers <int64> [ items <int64 > ]

List of SubCategory IDs the Drill belongs to.

related_drills
Array of integers <int64> [ items <int64 > ]

List of Drill IDs this Drill mentions.

Responses

Generated server url

http://localhost:5433/drill/id/{id}

Insert a New Drill.

Create a new Drill in the database with the given name. Returns the newly created Drill. To add categories, instructions, etc. use PUT /drill/id/{id} using the returned drill ID.

Request Body schema: application/json
required
name
required
string [ 1 .. 255 ] characters

Name of the Drill. Must be unique.

Responses

Generated server url

http://localhost:5433/drill

Retrieve Instructions by a Drill's ID.

Retrieve a list of a Drill's instructions using its ID. Returns only the description.

path Parameters
id
required
integer <int64>

Responses

Generated server url

http://localhost:5433/drill/id/{id}/how-to

Retrieve Instruction details by Drill ID and Instruction number.

Retrieve details of an Instruction using the Drill ID and the Instruction number. The Instruction number corresponds to its array position in the list of Instruction descriptions in /drill/id/{id}/how-to.

path Parameters
id
required
integer <int64>
number
required
integer <int64>

Responses

Generated server url

http://localhost:5433/drill/id/{id}/how-to/{number}
